常见的事件类型：
          鼠标：
              click          单击
              dblclick       双击
              contextmenu    右键菜单

              mouseover      进入（包括子节点）
              mouseout       离开（包括子节点）
              mousedown      按下
              mouseup        抬起
              mousemove      移动
              mouseenter     进入（不包括子节点）
              mouseleave     离开（不包括子节点）

          键盘：
              keypress       按键敲击
              keydown        按键按下
              keyup          按键松开

          文档：
              load           页面加载
              unload         页面关闭
              beforeunload   关闭之前

          表单：
              focus          获得焦点
              blur           失去焦点
              change         内容变化
              submit         表单提交
              reset          表单重置

          其他：
              scroll         页面滚动
              select         选中文字
              selectstart    开始选中内容时
              。。。
